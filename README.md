# JerryTechnical
Technical challenge for jerry.io


To install and run this package: 

1.  Clone package by running "git clone 

    $ git clone https://github.com/ManrajT/JerryTechnical.git  
2.  cd into folder
3.  Install dependencies

    $ npm install
    
4.  Run tests

    $ npm run test
